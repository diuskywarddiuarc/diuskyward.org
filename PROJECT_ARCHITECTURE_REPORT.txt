1. PROJECT OVERVIEW
========================================

DIU Skyward is a professional aerospace company website built with Next.js 16.2.1 and TypeScript. 
The project simulates a corporate aerospace platform with mission control systems, team management,
and content management capabilities.

Key Features:
- Professional aerospace company branding
- Mission Control Center (Admin Dashboard)
- Aerospace Operations Portal (Team Dashboard)
- Kanban task management with aerospace terminology
- Mission/news content management system
- Responsive design with aerospace theme
- Real-time dashboard updates
- Team collaboration tools

Theme: Professional aerospace corporation (similar to SpaceX, Blue Origin, NASA)
Target Audience: Aerospace engineers, mission controllers, corporate stakeholders
